# data/

`covtype_25k.csv.gz` — 20,000-instance stratified sample from UCI Forest Cover Type,
identical to the sample used in `report_1_sl` and `report_2_ol`.

Generated by `python run_pipeline.py` via `src/data_loader.py` on first run.
Seed: 7641. Stratified by Cover_Type (7 classes) using `StratifiedShuffleSplit`.
